These are stepwise instructions to run the project locally

To start backend

//change the current working directory to Backend

cd Backend

//install necessary packages

npm install

//to start server

node app.js


To start frontend

//change the current working directory to Backend

cd Phoenix-E-Mart

//install necessary packages

npm install

//To start frontend

npm run dev
